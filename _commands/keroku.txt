Clone the repository
Use Git to clone tensorflowjs-prueba's source code to your local machine.

$ heroku git:clone -a tensorflowjs-prueba
$ cd tensorflowjs-prueba
Deploy your changes
Make some changes to the code you just cloned and deploy them to Heroku using Git.

$ git add .
$ git commit -am "make it better"
$ git push heroku master
